# Reader Hub - System Design Masterclass

Reader Hub is a modern, high-performance web platform dedicated to deep-diving into the architecture of scalable systems. Curated and built by **Rajdeep Singh**, it provides a structured 30-day journey through system design concepts with a premium, focused reading experience.


## ✨ Key Features

- **30 Deep Dives**: A comprehensive curriculum covering everything from networking basics to complex microservices architecture.
- **Premium Glassmorphic UI**: A state-of-the-art design system that feels responsive, alive, and premium.
- **Distraction-Free Mode**: Toggle focus mode to hide all UI elements and focus entirely on the content.
- **Smart Reading Progress**: Real-time progress bar and estimated reading times for every article.
- **Dynamic Search & Filter**: Instant search and category-based filtering to find specific topics quickly.
- **Multi-Column Modern Footer**: A professional, organized footer with branding, navigation, and social links.
- **Dark/Light Mode**: Seamlessly switch themes with optimized contrast for different lighting conditions.
- **Accessibility First**: Responsive design, semantic HTML, and intuitive navigation.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS (Custom Variable-Based Design System)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Fonts**: Inter (Sans) & Playfair Display (Serif)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone [repository-url]
   ```
2. Navigate to the project directory:
   ```bash
   cd systemdesign_web
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## 📂 Project Structure

- `/src/App.jsx`: Main application logic, article data, and core components.
- `/src/index.css`: Comprehensive design tokens, layout styles, and animations.
- `/public/images/`: Highly optimized assets for system design visualizations.
